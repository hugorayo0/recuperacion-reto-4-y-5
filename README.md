# Restaurante Gourmet

# Descripción

Restaurante Gourmet es una aplicación web desarrollada con Flask para la gestión del personal de cocina de un restaurante.

El objetivo del proyecto es centralizar la información de los cocineros del establecimiento en un sistema sencillo y accesible. Actualmente, los datos del personal se gestionaban de forma manual, dificultando su consulta y mantenimiento.

Para resolver esta situación, se ha diseñado una aplicación web que permite registrar, visualizar y eliminar cocineros, almacenando toda la información en un archivo JSON estructurado.

# Objetivos del proyecto

- Centralizar la gestión de la información del personal de cocina.
- Facilitar el acceso a los datos de los cocineros de forma rápida y visual.
- Digitalizar el registro de especialidades y experiencia del personal.
- Aplicar buenas prácticas de desarrollo web con Python y Flask.

# Tecnologías utilizadas

- HTML
- CSS
- Python
- Flask
- JSON
- Jinja2
- Git

## Funcionalidades principales

### Gestión de cocineros

- Visualizar el listado completo de cocineros registrados.
- Añadir nuevos cocineros con nombre, especialidad y años de experiencia.
- Eliminar cocineros del sistema con confirmación previa.



## Base de datos

Los datos se almacenan en un archivo JSON con la siguiente estructura por registro:

- Nombre del cocinero
- Especialidad culinaria
- Años de experiencia

Este enfoque permite una persistencia de datos sencilla sin necesidad de un sistema de base de datos externo.

# Metodología de trabajo

El proyecto se ha desarrollado de forma individual aplicando un flujo de trabajo basado en el desarrollo incremental, añadiendo funcionalidades de forma progresiva y comprobando su correcto funcionamiento en cada etapa.

# Conclusión

Este proyecto ha permitido aplicar conocimientos de desarrollo web con Python, gestión de datos en formato JSON, diseño de interfaces con HTML y CSS, y estructuración de aplicaciones con el framework Flask en un contexto práctico y realista.
